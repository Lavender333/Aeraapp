# Quick Reference: First Responder

Role: `FIRST_RESPONDER`

## Live Incident Loop

- Review highest-priority unresolved safety signals
- Verify field status before updating records
- Escalate blocked access and medical-critical cases fast
- Hand off unresolved cases cleanly to recovery operations

## Never Do

- Mark estimated outcomes as verified
- Delay escalation for access-constrained households
- Close cases without clear status evidence
