# Scenario Drill: Hurricane Helion Lifecycle

Duration: 30–40 minutes
Audience: Mixed roles (Member, Single Org Admin, Network Org Admin)

## Objective

Run an end-to-end drill from warning to recovery while preserving role boundaries and communication quality.

## Phase Timeline

### Phase 1 — T-72 (Preparedness Sweep)

- Member verifies profile and household details.
- Single-org admin identifies vulnerable households.
- Network admin checks rollup and flags weakest child location.

**Success marker:** each role can state top 2 risks.

### Phase 2 — T-24 (Logistics Positioning)

- Member confirms latest accessibility/transport state.
- Single-org admin reviews local readiness constraints.
- Network admin validates location-by-location readiness posture.

**Success marker:** location-level priorities are documented.

### Phase 3 — Impact

- Member submits help/safety update via Help Wizard.
- Single-org admin triages incoming status signals.
- Network admin monitors cross-location impact trends.

**Success marker:** each action is role-appropriate and timely.

### Phase 4 — +24 to +72 Recovery

- Member updates condition changes.
- Single-org admin tracks unresolved local needs.
- Network admin compares recovery pace by location.

**Success marker:** clear recovery summary and next actions.

## Debrief Questions

- What decisions were delayed by missing data?
- Which broadcast instructions were ambiguous?
- Which role handoff had the most friction?
- What one process change should be adopted next drill?
